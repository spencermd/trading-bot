## CHART PATTERNS

### 1. 1-2-3 Pattern (S&P 500 Pattern)
```
The 1-2-3 pattern is a reversal pattern:

BULLISH 1-2-3:
1. Price drops to new low (1)
2. Price bounces up (2)
3. Price drops but NOT below (1)
4. Break above (2) = BUY

BEARISH 1-2-3:
1. Price rallies to new high (1)
2. Price drops (2)
3. Price rises but NOT above (1)
4. Break below (2) = SELL
```

### 2. Harmonic Patterns

#### ABCD Pattern
```
A → B → C → D
- AB = CD in distance
- Time symmetry
- D = Entry point
```

#### Gartley Pattern
```
- AB retraces 61.8% of XA
- BC retraces 38.2-88.6% of AB
- CD = 127.2-161.8% extension of BC
- D = Entry zone
```

#### Butterfly Pattern
```
- AB = 78.6% of XA
- BC = 38.2-88.6% of AB
- CD = 161.8-261.8% of BC
- More aggressive than Gartley
```

#### Crab Pattern
```
- AB = 38.2-61.8% of XA
- Extreme D point (161.8% extension)
- High reward:risk
```

#### Bat Pattern
```
- AB = 38.2-50% of XA
- BC = 38.2-88.6% of AB
- CD = 161.8-261.8% of BC
- D = Entry (接近 88.6% retracement)
```
