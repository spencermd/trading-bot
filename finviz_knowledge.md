# 📊 FINVIZ - Stock Screener & Market Overview

---

## WHAT IS FINVIZ?

**Financial Visualizations** - Free stock screener with visual charts and market data.

---

## KEY FEATURES

### 1. Market Overview
| Metric | Meaning |
|--------|---------|
| Advancing | Stocks going up |
| Declining | Stocks going down |
| Above SMA50 | In uptrend |
| Below SMA50 | In downtrend |

### 2. Stock Screener
Filter by:
- P/E ratio
- Market cap
- Sector
- Technical indicators
- Fundamentals

---

## SCREENER FILTERS

### Technical Signals:
| Filter | Meaning |
|--------|---------|
| **Top Gainers** | Biggest % gainers |
| **Top Losers** | Biggest % losers |
| **New High** | Breaking resistance |
| **New Low** | Breaking support |
| **Overbought** | RSI > 70 |
| **Oversold** | RSI < 30 |
| **Unusual Volume** | Volume spike |

### Fundamental Filters:
| Filter | Meaning |
|--------|---------|
| **P/E** | Price to Earnings |
| **EPS** | Earnings per share |
| **ROE** | Return on Equity |
| **Debt/Equity** | Leverage |
